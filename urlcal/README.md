# urlcal

Shared calendars by URL. No sign-up required. Live at urlcal.com.

## How it works

- Visit `urlcal.com/<name>` to create a calendar
- Claim it with a password
- Add events, share the URL

## Features

- Password-protected editing
- Week view with color-coded events
- Read-only sharing (share URL without password)
- No accounts, no tracking

## Status

This is an MVP and not production ready.
